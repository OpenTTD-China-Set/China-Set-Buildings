## 1 关于

本 NewGRF 为 OpenTTD 提供额外的建筑，专注于现代当代中国建筑。

## 2 基本信息

### 2.1 系统要求

- 最低测试通过的 OpenTTD 版本：12.0
- 不兼容 TTDPatch


### 2.2 安装

OpenTTD：
请参阅 https://wiki.openttd.org/NewGRF

本 NewGRF 可从游戏内在线内容下载。

## 4 联系方式
### 4.1 错误报告

请在以下地址报告您发现的任何错误
  错误跟踪器：https://github.com/OpenTTD-China-Set/China-Set-Buildings/issues

请始终包含错误的详细描述，最好附带截图和存档。同时请注明您使用的游戏版本以及本 NewGRF 的版本。

如果您的存档包含无法从 OpenTTD 在线内容获取的 NewGRF，请尝试在一个所有 NewGRF 都容易获取的新游戏中重现该错误。

如果您使用的是修改版游戏，请尝试在官方版本上重现该错误。如果无法重现，请勿在此报告，而应在该修改版的论坛主题中报告。

### 4.2 一般咨询

如果您有任何不能在 Github 上公开询问的问题，请通过私信联系 Ahyangyi，网址为 www.tt-forums.net。
